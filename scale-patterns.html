About Guitar Scale Patterns

How It Works

This application helps you visualize guitar scale patterns across the fretboard. Here's how to use it:

  • Key Selector: Choose the musical key for the scale patterns
  • Scale Type: Switch between Diatonic and Pentatonic scale patterns
  • Note Labels: Toggle visibility of note names on the fretboard
  • Scale Degrees: Toggle visibility of scale degree numbers
  • Root Highlight: Highlight major or minor root notes in the patterns
  • Fret Numbers: Numbers indicate the fret starting positions of the scale patterns on the fretboard

Study Suggestions

Practice these patterns to improve your fretboard knowledge:

  • Learn One Key at a Time: Master all patterns in a single key before moving to another
  • Connect the Patterns: Notice how patterns connect / overlap across the fretboard
  • Improvise: Use these patterns as a foundation for creating your own melodic lines

Tips

  • Use the note labels to learn the note names on the fretboard
  • Practice visualizing the patterns without looking at the screen
  • Notice that changing key moves the patterns to different positions on the fretboard
  • Combine patterns to create longer musical phrases using slides / postion shifts
